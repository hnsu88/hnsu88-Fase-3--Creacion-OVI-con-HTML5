HTML (HTML, HEAD, BODY)

Antes de poder iniciar con la escritura de código, es elegir un buen editor para el mismo. Existen múltiples opciones algunas muy básicas y otras mucho más completas; en mi opinión te recomiendo usar Visual Studio Code, el editor más usado por profesionales en la actualidad que puedes implementar sin importar que apenas estés comenzando.

Dentro de los documentos HTML, existen tres etiquetas fundamentales que le componen:

  • html: Es la raíz, en ella se contienen todos los elementos del documento que conformará tu página.
  • head: Contiene elementos de conexión o metadatos que no se muestran.
  • body: Contiene todos los elementos visibles de la página.

HTML5 (etiquetas, atributos y valores)

HTML es el lenguaje de marcado usado para la creación de sitios web, en este se define una estructura básica para la construcción y presentación de su contenido, el cual está estandarizado gracias a W3C o World Wide Web Consontium fundado en 11994 por Tim Berners-Lee quien es también el creador del mencionado lenguaje.

Propósito de CSS en el diseño de sitios web y cómo éste se implementa

CSS es un lenguaje de diseño gráfico, que nos permite definir y dar estilo a los componentes o elementos de un documento escrito en HTM, de manera que proporciona mucho más contenido visual a la interface de la página para hacerla más atractiva, aplicando una gran cantidad de propiedades y atributos que éste proporciona para lograrlo.
