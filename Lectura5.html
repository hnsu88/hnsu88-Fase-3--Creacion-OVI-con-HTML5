Link en HTML

Links encontramos en casi todas las paginas web, nos permite movernos entre ellas y buscar el contenido que querramos.

Hyperlinks

Los links en HTML los llamamos hyperlinks o hiperviculos, estos nos permite movernos entre documentos y los identificamos cuando pasamos el cursor y este cambia de forma o color.

La sintaxis para los links esta definida por la etiqueta < a > < /a > esta etiqueta tiene un atributo que especifica la direccion a la que va a llevar el hipervinculo esta etiqueta es href="" donde podemos poner la direccion del archivo o una URL externa.
