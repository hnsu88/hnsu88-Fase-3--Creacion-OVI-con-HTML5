Este espacio está diseñado para iniciarse en el mundo del diseño web, proporcionando información de lo más básico y fundamental para la creación de sitios web.

Objetivo :

Se pretende presentar una pequeña introducción de herramientas como HTML y CSS en la creación pura de sitios web, para conocer los aspectos básicos del diseño de sitios.

Objetivo específico 1:

Conocer los conceptos y estructura básica de HTML5 (etiquetas, atributos y valores)

Objetivo específico 2:

Conocer y entender los conceptos de la estructura básica página HTML (HTML, HEAD, BODY)

Objetivo específico 3:

Conocer el propósito de CSS en el diseño de sitios web y cómo éste se implementa.
