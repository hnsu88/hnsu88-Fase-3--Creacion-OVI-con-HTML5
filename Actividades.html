Actividad 1 creación de una plantilla

La primera actividad consiste en la creacion de una plantilla de documento basica en HTML, para esto seguimos los siguientes pasos:


  1. Vamos a usar cualquier editor de texto como por ejemplo el blog de notas.
  2. Como toda pagina HTML tiene una cabecera y un cuerpo donde pondremos el contenido de la pagina.
  3. Usamos las etiquesa; < html > < /html >, < head > < /head > y < body > < /body >
  4. La etiqueta < html > < /html >, nos indica el inicio de la pagina.
  5. La etiqueta < head > < /head >, nos indica donde ponemos el titulo de la pagina marcado con la etiqueta < time>< /time>.
  6. La etiqueta < body > < /body >, es donde ponemos el contenido de la pagina.
