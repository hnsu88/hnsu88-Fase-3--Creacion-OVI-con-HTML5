Fundamentos de HTML

HTML es las siglas de Hypertext Markup Language, es un lenguaje de marcado que se encarga de decirle al navegador (interprete) que desplegar y cómo hacerlo en una página web. HTML consiste en una serie de etiquetas que representan partes de un contenido. Entonces una página web es una serie de etiquetas anidadas.

Estructura básica de una página HTML5

La estructura básica de una página web en HTML5 consta de un < head > y un < body >, en el < head > es donde colocamos todas aquellas etiquetas (que no se van a visualizar), pero que sirven para indicar todas las características que el navegador necesita y en el < body > es donde se colocan todos los elementos que sí se van a mostrar y que forman el contenido de la página.
